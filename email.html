可以看到大多数时候垃圾邮件中出现的词汇都是比正常邮件中多的。这与垃圾邮件的绝对数量有关系（所以类别先验是必要的）。

可以过滤掉部分的停用词。

但又注意到 you, your 等单词可能对垃圾邮件区分有帮助，可能不能过滤。

我们将正常邮件中出现的多的词汇先过滤出来。

朴素贝叶斯的实现¶

这里的朴素贝叶斯，使用了 sklearn 上的公式进行实现。在不考虑很多数据上的普适性、错误、异常的情况下，实现起来还是比较简单的。

由于矩阵规模较大，且比较稀疏，所以可能额外增加了一定的成本。

In [90]:
from sklearn.metrics import accuracy_score
import numpy as np

class NaiveBayes:

    def __init__(self, alpha=0.1):
        self.alpha = alpha

    def fit(self, X, y):
        self.classes = np.unique(y)
        self.n_classes = len(self.classes)
        self.n_features = X.shape[1]
        self.n_samples = X.shape[0]

        separated_tests = [np.array(list(map(lambda x: x[0], filter(lambda y: y[1] == c, zip(X, y))))) for c in self.classes]
        self.class_log_prior = np.array([len(cls_) / self.n_samples for cls_ in separated_tests])

        self.feature_log_prior = np.array([data.sum(axis=0).toarray().reshape((self.n_features, )) for data in separated_tests])
        
        for i in range(self.n_classes):
            tot = self.feature_log_prior[i].sum()
            self.feature_log_prior[i] = np.log((self.feature_log_prior[i] + self.alpha) / (tot + self.n_classes * self.alpha))

    def predict_log_proba(self, X):
        return np.array([(self.feature_log_prior * x.toarray()).sum(axis=1) + self.class_log_prior for x in X])

    def predict(self, X):
        return np.array(list(map(lambda x: self.classes[x], np.argmax(self.predict_log_proba(X), axis=1))))

    def score(self, X, y):
        return accuracy_score(self.predict(X), y)
